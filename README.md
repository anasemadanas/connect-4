# Connect 4

React + Vite Connect 4 game.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## GitHub Pages

After pushing to the `master` branch, enable Pages in:
Repository Settings → Pages → Source: **GitHub Actions**.

Then the game should be available at:
https://anasemadanas.github.io/connect-4/
